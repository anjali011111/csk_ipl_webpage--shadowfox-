## 🛠️ Technologies Used

* HTML5 – For structuring the website content
* CSS3 – For styling, layout, animations, and responsive design
* JavaScript – For interactivity and dynamic functionality
* Bootstrap / Tailwind CSS – For responsive components and faster UI development
* Git & GitHub – For version control and project management
* Netlify – For deployment and hosting

## 💡 How I Built This Website

I started by designing the layout and structure of the website using HTML. Then I styled the pages with CSS using a CSK-inspired yellow and orange theme to create an attractive user interface.

JavaScript was used to add interactive features such as navigation effects, smooth scrolling, and dynamic sections. I also implemented responsive design techniques to ensure the website works properly on both mobile and desktop devices.

The website includes multiple sections such as:

* Home page
* Player profiles
* Match schedules
* Team achievements
* Fan engagement section
* Special highlight section for MS Dhoni

Finally, I used GitHub for managing the source code and deployed the project using Netlify for public access.
